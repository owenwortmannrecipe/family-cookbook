🔧 One-time Firebase setup needed

Open index.html in a text editor and replace the placeholder values in the firebaseConfig block near the top with your own Firebase project credentials.

How to get them (free, takes ~3 minutes):
1. Go to console.firebase.google.com → Create project
2. Add a Web app → copy the config object
3. In the left menu: Firestore Database → Create database → Start in test mode
4. Paste your config into this file and you're done — all family devices sync instantly!
